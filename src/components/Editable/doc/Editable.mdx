# Editable

This component control the inline edition for the element passed as child. The element(s) passed as children only will be visible
when are in edit mode after edition the value will be displayed by this component.

## How to use

```javascript
import { Editable } from "@outsystemsui";

const MyEditableInputComponent = () => {
    const inputRef = useRef();
    const [task, setTask] = useState("");

    return (
        <Editable
            text={task}
            placeholder="Write a task name"
            childRef={inputRef}
            type="input"
        >
            <input
                ref={inputRef}
                type="text"
                name="task"
                placeholder="Write a task name"
                value={task}
                onChange={e => setTask(e.target.value)}
            />
        </Editable>
    );
};

export default MyEditableComponent;
```
