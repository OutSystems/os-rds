# Overlay

This component will can be used to overlay two components.
It is possible to define a custom position passing a class with the position or pick a base position.

## How to use

```javascript
import { Overlay, OverlayPosition } from "@outsystemsui";

const MyComponent = () => (
    <Overlay position={OverlayPosition.TopLeft} overlay={**Decoration Component**}>
        {**Component to Decorate**}
    </Overlay>
);

export default MyComponent;
```
